
Agent-based model shows genetic drift drives Prochlorococcus genome reduction

=== Appendix ===

IAMvl2: The main program for ABM-based genome simulation.

IAM_I_g_0001.txt: The genomic sequence used in the study (NCBI RefSeq database; Accession number GCF_000011465; Prochlorococcus marinus MED4).

IAM_I_r_0001.txt: A modified genomic feature file (GFF) generated by Prodigal.

IAM_I_c.txt: The configuration file for genome simulation, in which the manually adjusted parameters are shown as follows:
	“dimnnt”: the length of genomic sequence
	“dt”: division time
	“Popic”: effective population size (Ne)
	“npGLO”： the number of simulated genome
	“RSi” and “RSj”: the random seed number
	“Vmax0C”, “Vmax0N” and “Vmax0P”: nutrient availability of C, N, and P

IAM_I_t.txt: An essential time-variable input file that does not apply to this study.
